Tool description

The tool visualizes how the mergeMap, switchMap, concatMap and exhaustMap operators work.

  • The user can select any combination of the operators.
  • Tool can either be in on or off state (controlled by xhr and reset buttons).
  • Xhr event consists of three steps and complete message, separated by one second intervals (xhr-steps stream).
  • Tool counts times the xhr button has been clicked (xhr-counter stream).
  • The flattening strategies, which have been checked, react to the xhr-counter stream and return combined xhr-counter stream and xhr-steps stream (strategy stream).
  • Strategy streams are subscribed by functions responsible for rendering the logs on the screen.

Enjoy learning the flattening strategies!
